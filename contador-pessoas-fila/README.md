# Contador de Pessoas em Fila

Este projeto utiliza técnicas de visão computacional para contar o número de pessoas em uma fila a partir de um vídeo.

## Funcionalidades

- Seleção de região de interesse (ROI) representando a área da fila
- Detecção de pessoas utilizando o detector HOG do OpenCV
- Contagem em tempo real do número de pessoas na fila
- Exibição visual do resultado no vídeo
